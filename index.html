Command-line Toolset for Bisulfite Sequencing Data Analysis

Advanced Features

1. Unified data format

use standard CGmap/ATCGmap formats as central format, for ease of sharing methylomes. ATCGmap/CGmap could be derived from BAM files, which are produced by BS-Seeker2 or Bismark

2. Command-lines convenient for parallel processing and to be extended

  • Implemented as command-lines, run on both Linux and MAC
  • Support STDIN and STDOUT, so that commands can be joined using pipe

3. Support instantly retrieving by designed binary file formats

  • binary format CGbz for CGmap
  • binary format ATCGbz for ATCGmap

4. Use dynamic fragmentation strategy for DMR analysis

  • useful for low-coverage (WGBS) and fragmented (RRBS) DNA methylomes

5. Novel strategies for SNV calling

  • introduced ambigous genotype prediction
  • significantly improve the precision especially in heterozygous SNVs

6. Support ASM analyses

  • use precise heterozygous SNVs as input
  • designed Tanghulu plots for showing allele-specific DNA methylation on reads

7. Advanced methylation analysis and visualization

provide user-friendly functions for visualizing methylomes at multiple levels, such as design a Tanghulu plot for visualizing methylation status on original reads, and design a Lollipop plot to reveal both lowly-covered cytosines and un-methylated cytosines in a local region


Unified data format [ CGmap and ATCGmap ]

- Facilitate seamless data analysis and sharing using unified data format.
- Allow fast subdata fetching using compressed and indexed CGbz and ATCGbz format.
- Offer multiple tools for file manipulation and format coversion.

Enhanced performance on SNV calling

- Provide two methods based on bayesion and binomial models, respectively.
- Improve prediction precision by considering vague reads due to bisulfite coversion.
- Accelerate allele-specific methylation analysis.

Advanced methylation analysis and visualization

- Seamless analysis pipeline for WGBS (MethylC or PBAT) and RRBS data.
- Optimized presentation of methylation data.
